Load this folder as a project in Unity 3D. 
The recommand version is 2018.4.14f1.

Tonkin Chess:
	1. Select a unplaced chess by mouse click, and place it at an empty intersection on board by click again.
	2. When all the chesses are placed, select a chess and move it to an adjacent intersection which is empty.

	Whenever all intersections of a line is occupied by the chesses with same color, the owner wins.
